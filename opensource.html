Tools for petrology

Here you can find the scripts or tools I developped myself or with colleagues.

1. Li W & Zhang Y. PyAp: a python package for calculating magmatic volatile, trace element concentrations, and oxygen fugacity using mineral apatite. Code available at pyAp.

2. Mass balance calculation using non-negative and matrix decomposition algorithms, with MCMC propagating errors on phases and bulk composition. Code available at MassBalanceCal.

3. Multi-component olivine diffusion chronometry, with uncertainties integrated. Developped with Olivier Namur, T.D. Van Gerve (python). Please contact me for detail using and discussion.

4. Stepwise backward F-test for multiple linear regression (python). Contact me for details.

5. A script to convert alphaMELTS outputs to formatted spreadsheet.
